《仙剑奇侠传三》分辨率补丁 + 配音插件 | 自述文件


分辨率补丁主要功能：
   提高游戏分辨率（完美修正界面错位问题）
   修正游戏切屏崩溃问题
   内置免 CD 补丁，无需修改注册表
   内置窗口化补丁，无需其他工具（窗口化时按 F12 可解除鼠标锁定）
   内置语言切换选项，无需改区工具
   可以替换游戏字体
   可以调出游戏程序内置的战斗编辑器
   可以加载外部配音插件


配音插件主要功能：
   可以对剧情对话、诗词、过场动画、战斗对话进行配音
   此外还可以使用脚本进行显示图片、播放视频等复杂操作
